## The Jukebox — Data Sonification Experience

**Concept:** Turn existing datasets into playable music tracks using the Web Audio API. A pixel-art jukebox UI where each "record" is a real dataset from the site — deer deaths become a bassline, weather patterns become ambient pads, baseball stats become drum sequences.

**Why it's compelling:**
- Radically innovative — data sonification on the web is almost nonexistent
- Perfectly thematic — a jukebox is the iconic dive bar object
- Accretive — cross-pollinates every other page's data
- Technically elegant — Web Audio API, no dependencies, no API keys
- Ambient potential — leave it running like the Weather Desk
- Reveals hidden patterns — sonification surfaces temporal patterns charts miss

**Technical approach:** Web Audio API (built into browsers), vanilla JS, no build tools. Each dataset maps to musical parameters — values become pitch, time becomes rhythm, categories become instruments.
